Welcome to php-dynamodb Docuemntation!
****************************************

php-dynamodb is a PHP library that can be used to interact with Amazon DynamoDB. It provides a layer of abstraction between your code and the DyanamoDB-related classes made available by the AWS SDK for PHP.
